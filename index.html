WHY THIS IS NEEDED

Browsers block direct calls to api.anthropic.com from pages on GitHub Pages, Netlify, etc. due to CORS policy. A small proxy worker sits in between and adds the required headers.

CLOUDFLARE WORKER SETUP (FREE · 3 MIN)

  1. 1.Go to workers.cloudflare.com → sign up free
  2. 2.Click Create Worker → paste the contents of cloudflare-worker.js → Deploy
  3. 3.In Worker Settings → Variables & Secrets → add Secret:
    ANTHROPIC_API_KEY = your key from console.anthropic.com
  4. 4.Copy your worker URL (e.g. https://aura-proxy.your-name.workers.dev) and paste below
